🔑 Generador de tokens permanentes de Facebook (PPF)

Todo ocurre en tu navegador. Nada se envía a terceros salvo la propia API de Facebook. Genera tokens de Página que no caducan.

Antes de empezar:
  1. Abre el Explorador de la API Graph, app PPF Auto Publish, y copia el Token de acceso (empieza por EAA…). Dura ~2 h: úsalo pronto.
  2. Ten a mano la Clave secreta de la app (Ajustes → General → Afficher).
